# Анализатор логов J1939

Гибкий инструмент на языке С для анализа и исследований логов CAN-J1939 

## Функционал 

- Код программы не является стабильным
- Он постоянно переписывается , модифицируется и перекомпилируется под конкретную аналитическую задачу
- Это ядро анализатора

## Примеры

- Какие блоки в сети (лог по сути сеть)
- Колличество сообщений с каждого блока
